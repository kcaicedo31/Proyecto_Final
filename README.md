# SportShop - Tienda Deportiva

Tienda online de productos deportivos con carrito de compras y sistema de pago.

# Integrantes
- Kevin Caicedo
- Santiago Vela

## Características

- Catálogo de productos deportivos
- Sistema de búsqueda y filtros
- Carrito de compras interactivo
- Proceso de pago con tarjeta y efectivo
- Gestión de inventario

## Estructura

```
Proyecto_Final/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
├── Images/             # Imágenes de productos
└── README.md           # Este archivo
```

## Uso

1. Abrir `index.html` en el navegador
2. Navegar por los productos
3. Agregar productos al carrito
4. Proceder al checkout

## Productos Disponibles

- **Calzado**: Nike, Adidas, Puma, New Balance
- **Ropa**: Camisetas, sudaderas, shorts, leggings
- **Equipamiento**: Pelotas, raquetas, guantes, bicicletas
- **Fitness**: Mancuernas, bandas elásticas, esterillas, kettlebells
